What is this
===
A fizzbuzz application and API, why? `¯\_(ツ)_/¯`

Requirements
===

* Bundler 10.x
* Ruby 2.x

Set up
===

```bash
$ bundle install
$ rake db:setup db:migrate
$ rails s
```

Running tests
===

```bash
$ rspec spec
```

Accessing the app
===

* For the UI: `open http://localhost:3000`
* For the API: `open http://localhost:3000/numbers`
